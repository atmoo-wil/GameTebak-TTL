Selamat Datang di Website Nge-date

Nge-Date adalah game unik yang dapat menebak tanggal, bulan, tahun bahkan hari lahir kamu. Game ini merupakan hasil improvisasi dari seorang mahasiswa salah satu universitas terkenal di Indonesia.

Rules

  • Isi nama kamu
  • Mengklik OK jika ada angka lahirmu
  • Mengklik Cancel jika tidak ada angka lahirmu
  • Program akan menebak hari lahirmu dalam hasil akhir
  • Fokus, karena jika angka yang dipilih salah maka akan salah
  • Game ini hanya berfungsi pada browser chrome
  • Ikuti peraturan di atas dengan baik.
